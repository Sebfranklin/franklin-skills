# Franklin's Agent Skills

A collection of Agent Skills for Gemini CLI, OpenCode, and other AI agents, following the [agentskills.io](https://agentskills.io) specification.

## Installation

To install these skills globally for all supported agents:

```bash
npx skills add Sebfranklin/franklin-skills --all -g
```

## Available Skills

This repository contains skills for:
- Development (Python, JS, Playwright, Testing)
- Content Creation (Image-to-Video, Video Studio)
- Productivity (Obsidian, GWS Docs, Document Creation)
- System Management (Skills Manager, CI/CD)
- And more.
